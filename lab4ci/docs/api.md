# API Documentation

## Sales Aggregator API

### Описание

Sales Aggregator - веб-сервис для поиска, публикации и оценки студенческих скидок.

### Основные эндпоинты

#### Пользователи

- `GET /users/register/` - Регистрация нового пользователя
- `GET /users/login/` - Вход в систему
- `GET /users/logout/` - Выход из системы

#### Посты о скидках

- `GET /` - Главная страница со списком постов
- `GET /create-post/` - Создание нового поста (требует авторизации)
- `POST /vote/<post_id>/<vote_type>/` - Голосование за пост (up/down)
- `POST /toggle-favorite/<post_id>/` - Добавление/удаление из избранного
- `GET /my-posts/` - Мои посты (требует авторизации)
- `POST /delete-post/<post_id>/` - Удаление поста (требует авторизации)
- `GET /favorites/` - Избранные посты (требует авторизации)

### Фильтрация и сортировка

- `?category=<category_id>` - Фильтрация по категории
- `?sort=rating` - Сортировка по рейтингу
- `?sort=newest` - Сортировка по дате (по умолчанию)

### Модели данных

#### Post
- `title` - Заголовок поста
- `description` - Описание условий скидки
- `place` - Место (магазин, кафе и т.д.)
- `category` - Категория
- `author` - Автор поста
- `rating` - Рейтинг (вычисляемое свойство: upvotes - downvotes)

#### Category
- `name` - Название категории

#### Vote
- `post` - Пост
- `user` - Пользователь
- `vote_type` - Тип оценки ('up' или 'down')

#### Favorite
- `user` - Пользователь
- `post` - Пост
